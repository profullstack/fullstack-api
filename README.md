# fullstack-api

Fullstack API starter kit with Koa, rethinkdb and websockets.

## install
	
	git clone <url>
	npm i
	npm start

## production

    # locally
    ./bin/deploy.sh
    
    # on server
    npm run start:production

## generate certs

Self-signed certs:

    ./bin/certs

## requirements

MacOS:

	- xcode is required (see App Store)
